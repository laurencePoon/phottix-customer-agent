# Phottix Customer Agent

Local customer analysis, product recommendation, and outreach email tool.

## How to Use on Windows

1. Download this project from GitHub.
2. Extract the ZIP file.
3. Install Node.js if needed: https://nodejs.org/
4. Double-click `start.bat`.
5. A black CMD window will open.
6. Wait until you see `listening on http://127.0.0.1:8787`.
7. Keep the CMD window open while using the app.
8. Open this link in Chrome or Edge:

```text
http://127.0.0.1:8787/
```

## Notes

- `127.0.0.1` is local. Each user must run `start.bat` on their own computer.
- Do not test your friend's `127.0.0.1` from Codex or from another computer. It only works on the same computer where `start.bat` is running.
- If the server window is closed, the website will stop working.
- Excel import/export may require Python and `openpyxl`.

```cmd
pip install openpyxl
```
