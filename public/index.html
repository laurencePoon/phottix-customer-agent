產品資料庫 / Product Database

Recommendation Pool 是主動推薦池，Full Catalog 是完整查詢庫。

Recommended For： A = 批發商 / Wholesaler, B = 實體零售商 / Physical Reseller, C = 網店零售商 / Online Reseller, D = 工作室 / Studio or End User, All = 所有客戶皆適用 / suitable for all buyers.
Priority： 1 = 最優先推薦 / first recommendation, 2 = 第二優先 / second, 3 = 第三優先 / third, 空白或較大數字 = 排在後面 / blank or larger numbers appear later.
設定邏輯 / Setup logic： Recommended For 決定「這個產品給誰」；Priority 決定「符合的人裡，先推哪個」。
建議設定 / Suggested setup： 高單價 / 批發量大產品 = A + Priority 1； 實體店容易展示的熱銷款 = B + Priority 1； 輕便、容易網店銷售的產品 = C + Priority 1； 套裝、服務型、工作室常用產品 = D + Priority 1； 基礎配件、所有人都可能需要 = All + Priority 2 或 3。
